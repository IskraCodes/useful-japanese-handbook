# Nick's Useful Japanese Handbook

Single-file, mobile-friendly Japanese conversation handbook.

The published GitHub Pages site serves `index.html` directly. The app is offline-capable once loaded and stores custom words/favorites in the browser's localStorage.

## Data Notes

- Built-in entries are embedded in `index.html`.
- Custom entries are stored only in the browser unless exported.
- Use the app's JSON export/import controls to move custom entries between browsers/devices.
